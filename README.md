# 🌱 Soil Moisture Prediction using Deep Learning

## 📌 Overview
This project predicts whether a plant needs water based on environmental conditions.

## 📊 Features Used
- Soil Moisture
- Temperature (°C)
- Sunlight Hours

## 🧠 Model
- Neural Network built using TensorFlow/Keras
- Binary Classification

## ⚙️ Workflow
1. Data preprocessing
2. Feature scaling
3. Train-test split
4. Model training
5. Evaluation

## 📈 Results
- Achieved basic accuracy on small dataset
- Demonstrates deep learning pipeline

## ⚠️ Limitations
- Very small dataset (~15 samples)
- Deep learning not ideal for small data

## 🚀 Future Improvements
- Use larger real-world dataset
- Compare with ML models (Logistic Regression, Random Forest)
- Hyperparameter tuning

## 🛠 Tech Stack
- Python
- Pandas
- TensorFlow / Keras
- Matplotlib
